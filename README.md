<h1>Scrapper with Nokogiri</h1>

This project use:

    </li>
    <li>Nokogiri: Nokogiri parse HTML - <a href='http://www.nokogiri.org/'>http://www.nokogiri.org</a>

    </li>

    </li>
</ul>
